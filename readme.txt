# discouille
Un nouveau thème pour BetterDiscord !
par Bavouille

comment installer ce thème ?

1° Installer BetterDiscord (https://urlz.fr/jIg4)
2° Ouvrir le fichier thèmes (pour le trouver , ouvrir Discord -> réglages -> BetterDiscord/thèmes -> Bouton "ouvrir dossier thèmes")
3° Mettre le fichier discouille-V1.0.css dans le dossier ouvert

Merci d'avoir installé Discouille !
